# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| 1.2.x   | :white_check_mark: |
| < 1.2.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **DO NOT** open a public issue.
Instead, please send an email privately to: **[INSERT CONTACT EMAIL HERE]**

Please include the following in your report:
- Type of issue (e.g., buffer overflow, memory leak, unsafe string parsing)
- Full paths of file(s) where the vulnerability exists
- Any special configuration required to reproduce the issue
- Step-by-step instructions to reproduce the issue
- Proof-of-concept or exploit code (if possible)

We will respond within 48 hours to acknowledge the report and outline the next steps for patching the vulnerability.
